## Tools

### `encode_base64`
Encode text to Base64.

**Parameters:**
- `text` (str)

### `decode_base64`
Decode Base64 to text.

**Parameters:**
- `encoded` (str)

### `encode_url`
URL-encode a string.

**Parameters:**
- `text` (str)

### `encode_html`
HTML-encode special characters.

**Parameters:**
- `text` (str)

### `to_hex`
Convert text to hexadecimal.

**Parameters:**
- `text` (str)
